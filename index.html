The Basics of the Web and HTML


What is A Web Page?

A webpage is a text document written in a language called HTML. HTML stands for hyper text mark up language which is the main document type of the web.


How the Web Works?

The web has four components - the Browser, Internet, HTTP (which is the main protocol of the web), and Servers. When a user open a web browser and connects to www.udacity.com by sending an HTTP request via internet, the server for www.udacity.com will then retrieves the document for the website and sends it back to the user's browser. HTML Vocabulary

The Basic HTML vocabulary consists of Tags, Elements and Attributes. Most Tags have an opening and a closing tag. Some tags (called "void" tags) do not require a closing tag. An HTML element refers to everything within a set of opening and closing tags. The Attribute is a property of an HTML element. This video will show you the structure of the HTML tags and elements as well as to make links using attributes.


Inline vs. Block Elements

HTML elements are either inline or block. Block elements create an "invisible box" around the content inside of them. a, span, br, tag and strong are all inline elements while div, p and form are block elements.
